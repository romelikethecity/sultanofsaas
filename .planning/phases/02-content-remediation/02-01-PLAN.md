  <action>
For each of the 3 content files, find and fix all banned word and em-dash violations:

**Banned word replacements (context-dependent, not mechanical):**
- "genuinely" -> delete or rewrite: "genuinely impressive" -> "impressive", "genuinely useful" -> "useful" or "worth using"
- "truly" -> delete or rewrite: "truly stands out" -> "stands out"
- "really" -> delete or use a specific: "really fast" -> "fast" or give the number
- "actually" -> delete or rewrite: "actually works" -> "works", "you actually get" -> "you get"
- "seamless" -> "smooth", "effortless", "clean", or describe the specific UX (but NEVER replace when referring to "Seamless.AI" the tool)
- "frictionless" -> "smooth", "clean", "painless"
- "best-in-class" -> "top-tier", "leading", "strongest in category", or be specific about what makes it best
- "robust" -> "solid", "capable", "full-featured", or name the specific capability
- "leverage" -> "use", "tap into", "build on"
- "cutting-edge" -> "modern", "advanced", or name the specific technology
- "game-changer" -> name the specific impact
- "end-to-end" -> "full-stack", "complete", "start-to-finish", or name what it covers

**Em-dash replacements:** Replace unicode \u2014 with the appropriate punctuation:
- Parenthetical asides -> use commas or parentheses
- Lists/elaboration -> use a colon
- Sentence breaks -> use a period and start new sentence
- Contrast -> use "but" or a semicolon

**False reframe (trello / stage_guidance.enterprise in tools_pm.py):**
Find the "not X, it's Y" pattern and rewrite to state the point directly. Preserve the original meaning. Example: "It's not a project tool, it's a board tool" -> "Trello is a board tool." Just say what it IS.

**Voice rules:** Maintain The Sultan's voice -- opinionated, direct, uses contractions, backs opinions with evidence. Don't make the text sound generic or corporate.
